# 财税短视频获客系统工作台 - 设计规范

## 1. 概念与愿景

一个专为财税行业打造的高端短视频获客工作台，采用深色商务风格，融合数据仪表盘美学与专业财税质感。整体氛围沉稳、专业、高效，让财税从业者感受到工具的专业性与可信度。模块命名采用中国古代意象（特劳特、灵犀、千里眼、东风），赋予系统文化底蕴与品牌识别度。

## 2. 设计语言

### 色彩系统
- **主背景色**: `#0D0F14` (深空黑)
- **卡片背景**: `#161A22` (暗夜灰)
- **边框色**: `#2A303C` (炭灰色)
- **主色调**: `#3B82F6` (科技蓝 - 代表专业与信任)
- **辅助色**: `#10B981` (翡翠绿 - 代表增长与盈利)
- **警示色**: `#F59E0B` (琥珀金 - 财税金)
- **强调色**: `#8B5CF6` (紫罗兰 - 高端感)
- **文字主色**: `#F1F5F9` (银白)
- **文字次色**: `#94A3B8` (灰银)

### 字体系统
- **标题字体**: "Noto Sans SC", system-ui, sans-serif (中文优化)
- **正文字体**: "Inter", "Noto Sans SC", system-ui, sans-serif
- **数据字体**: "JetBrains Mono", monospace (数字与代码)

### 空间系统
- 基础间距单位: 4px
- 卡片内边距: 24px
- 组件间距: 16px
- 模块间距: 32px

### 动效哲学
- 模块切换: 淡入淡出 + 轻微位移, 300ms ease-out
- 按钮交互: scale 0.98 on press, 150ms
- 数据展示: 数字递增动画, 1000ms
- 卡片悬停: 边框发光效果, 200ms

### 视觉资产
- 图标库: Lucide React (与 shadcn/ui 一致)
- 装饰元素: 微妙渐变光晕、网格背景纹理
- 模块图标: 各模块专属配色图标

## 3. 布局与结构

### 整体架构
```
┌─────────────────────────────────────────────────────┐
│  顶部状态栏 (Logo + 系统名称 + 当前时间)              │
├────────────┬────────────────────────────────────────┤
│            │                                        │
│  左侧导航  │           主内容区域                     │
│  (固定)    │    ┌─────────────┬────────────────┐   │
│            │    │  表单填列区  │  结果展示区     │   │

│  图标+名称 │    │             │               │   │
│            │    └─────────────┴────────────────┘   │
│            │                                        │
```

### 导航结构
1. 特劳特 (选题生成) - 灯泡图标 - 蓝色
2. 灵犀 (脚本生成) - 文件文本图标 - 绿色
3. 小丽 (视频生产) - 视频图标 - 紫色
4. 千里眼 (数据监控) - 图表图标 - 金色
5. 东风 (私域运营) - 用户图标 - 红色

### 响应式策略
- 桌面端 (≥1280px): 完整双栏布局
- 平板端 (768-1279px): 导航收起为图标模式
- 移动端 (<768px): 底部 Tab 导航

## 4. 功能模块设计

### 4.1 特劳特 - 选题生成
**表单区**:
- 账号选择: 单选按钮组 (主号/副号)
- 内容类型: 多选下拉 (税务筹划/政策解读/案例分析/合规提醒/节税技巧)
- 生成数量: 数字输入 (1-20)
- 行业关键词: 文本输入 (可选)

**结果区**:
- 选题卡片列表, 每张包含: 序号、选题标题、预估时长、热度指数、适用场景
- 一键复制功能
- 导出功能

### 4.2 灵犀 - 脚本生成
**表单区**:
- 选题选择: 从选题库选择或手动输入
- 视频时长: 单选 (30秒/60秒/90秒)
- 风格选择: 多选 (专业严肃/轻松幽默/故事叙述/专家访谈)
- 核心卖点: 文本输入
- CTA话术: 文本输入

**结果区**:
- 分段脚本展示, 包含:
  - 时间轴 (00:00-00:10 开场钩子)
  - 情绪标注 (平静→激动→共鸣)
  - 话术内容
- 钩子话术高亮显示
- 一键复制完整脚本

### 4.3 小丽 - 视频生产
**表单区**:
- 脚本来源: 选择已有脚本
- 数字人选择: 下拉选择 (已克隆数字人列表)
- 背景模板: 网格选择 (商务/简约/科技)
- 字幕样式: 下拉选择
- BGM配置: 下拉选择 + 音量滑块

**结果区**:
- 生产排程日历视图
- 批量任务列表
- 配置模板预览

### 4.4 千里眼 - 数据监控
**表单区**:
- 视频数据录入表单:
  - 视频标题、发布日期、播放量、点赞数、评论数、转发数、引流私域数
- 日期范围选择

**结果区**:
- 核心指标卡片 (总播放/总转化/转化率)
- 数据表格 (支持排序筛选)
- 转化率趋势图表 (简易柱状图)
- 优化建议卡片

### 4.5 东风 - 私域运营
**表单区**:
- 功能选择 Tab:
  - 评论回复 SOP
  - 私信话术
  - 客户分层
  - 产品转化

**结果区** (根据选择展示):
- 评论关键词匹配规则 + 回复模板
- 3段式私信话术模板
- 客户分层矩阵 (潜在/意向/高价值/成交)
- 4类产品转化话术库

## 5. 组件规范

### 导航项
- 默认: 透明背景, 图标+文字灰色
- 悬停: 背景 rgba(59,130,246,0.1), 图标变蓝
- 激活: 左侧蓝色边框, 背景 rgba(59,130,246,0.15), 图标文字高亮

### 按钮
- 主按钮: 蓝色渐变背景, 白色文字
- 次按钮: 透明背景, 蓝色边框
- 危险按钮: 红色边框/背景

### 卡片
- 背景: #161A22
- 边框: 1px solid #2A303C
- 圆角: 12px
- 悬停: 边框变为 #3B82F6, 添加微妙阴影

### 表单输入
- 背景: #0D0F14
- 边框: #2A303C
- 聚焦: 边框变为蓝色, 添加发光效果

### 数据展示
- 数字: 使用等宽字体, 添加千位分隔符
- 百分比: 绿色(正向) / 红色(负向)
- 变化趋势: 带箭头指示

## 6. 技术实现

### 状态管理
- React useState/useReducer 管理本地状态
- localStorage 持久化数据

### 数据存储结构
```typescript
interface Topic {
  id: string;
  title: string;
  account: 'main' | 'secondary';
  type: string[];
  duration: number;
  heatIndex: number;
  createdAt: string;
}

interface Script {
  id: string;
  topicId: string;
  content: ScriptSegment[];
  duration: number;
  style: string[];
  hookPhrase: string;
  cta: string;
}

interface VideoRecord {
  id: string;
  title: string;
  publishDate: string;
  views: number;
  likes: number;
  comments: number;
  shares: number;
  privateConversions: number;
}
```

### 关键实现
- 模块懒加载, 减少首屏体积
- 表单验证使用 React Hook Form
- 数字动画使用 CSS counter + keyframes
- 深色主题全局 CSS 变量

## 7. 设计禁忌

- 避免过于花哨的渐变或动效, 保持财税专业感
- 不要使用粉嫩或卡通风格的配色
- 避免过多圆角, 保持硬朗商务线条
- 不要使用未授权的中文字体
